# Ariadne
## Helper for web-based escape rooms

Ariadne is a helper for web-based enigma games such as notpr0n or Mort-Vif.

## Features
- Tree structure for saving steps
- Automatic Username-Password entry for solved puzzles

## Dependencies
- support for C++23
- raylib 6.0
- rlImGui
- Dear ImGui
- pugixml
